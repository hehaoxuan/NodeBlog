# NodeBlog
## 项目简介
### 基于Node.js+Express+MongoDb+Mongoose+Ejs的博客，实现新增、删除、路由、详细页功能
## 安装依赖
### ```npm init```
## 项目启动(在项目路径下)
### ```nodemon server```
## 端口监听
### ```http://localhost:3001```
